# MLN131 Quest - Chủ nghĩa xã hội khoa học

Ứng dụng React/Vite học môn MLN131 với dữ liệu viết cứng, biên soạn theo **Giáo trình Chủ nghĩa xã hội khoa học** dành cho bậc đại học hệ không chuyên lý luận chính trị, NXB Chính trị quốc gia Sự thật, 2021.

## Nội dung

- 7 chương đúng cấu trúc giáo trình
- 21 bài học tóm tắt
- 24 flashcard
- 35 câu hỏi trắc nghiệm có giải thích
- Quiz theo chương, thi thử, mini game
- MLN Tutor offline
- Ghi chú, XP, huy hiệu và localStorage

## Chạy dự án

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
